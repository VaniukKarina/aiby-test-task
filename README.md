# AIBY Front-end Test Task
This project is a full-screen WebView banner for iPhone SE, 8, 8+, X, and 12.

## Tech stack
- HTML5
- CSS3
- JavaScript

## Features
- Fixed viewport layout for iOS WebView
- Language switch via `?lang=`
- Language-based typography adjustments
- Device-specific layout handling
